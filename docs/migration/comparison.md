# GPTers 사이트 마이그레이션 비교

Bettermode SaaS 기반 → Next.js 자체 플랫폼 전환 과정에서의 변화를 기록합니다.

## 플랫폼 전환 개요

| 항목 | As-Is (Bettermode) | To-Be (Next.js) |
|------|-------------------|-----------------|
| 플랫폼 | Bettermode SaaS | Next.js 16 (App Router) 자체 플랫폼 |
| 디자인 시스템 | 없음 (인라인 커스텀) | @gpters-internal/ui 토큰 + shadcn/ui |
| 스타일링 | Bettermode 위젯 기반 | Tailwind CSS v4 (CSS-first) |
| 어드민 | Bettermode 관리 + Retool + 에어테이블 | 통합 어드민 대시보드 |
| LMS | 외부 LMS 링크 이동 | 사이트 내 LMS 내재화 |
| DB/인프라 | Bettermode 종속 | Supabase (auth, DB, storage) |
| 검색 | Bettermode 기본 (부정확) | 통합 검색 (게시글 + 스터디 + 사용자) |
| 디자인 방향 | 위젯 나열형 | Reddit-inspired (투표 큐레이션, Compact 카드) |

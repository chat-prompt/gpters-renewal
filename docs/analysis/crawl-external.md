## 1. AI 스터디 목록 (/ai-study-list)

### 기본 정보
- **URL**: https://www.gpters.org/ai-study-list
- **역할**: AI 스터디 프로그램 목록 (핵심 전환 페이지)
- **렌더링**: 완전 CSR (스터디 카드 데이터는 GraphQL API로 동적 로드)

### 프로모션
- 상단 배너: "단 7일, 가장 좋은 혜택! AI 스터디 21기 슈퍼 얼리버드 START"
- 스터디 기수제 운영 (21기 모집 중)

### 스터디 카드 구조 (추정)
- 스터디명
- 설명/커리큘럼
- 기간
- 가격 (얼리버드 할인)
- 강사/스터디장 정보
- 모집 상태 (모집중/마감)
- 신청 CTA

### 기술 정보
- Bettermode 내부 스페이스로 구현
- 스페이스 slug: `ai-study-list`
- AI 캠프 컬렉션 하위

### 한계
- SPA 구조로 스터디 카드 상세 데이터(가격, 커리큘럼 등)는 정적 크롤링 불가
- 실제 카드 수, 카테고리 필터 여부는 런타임 확인 필요

## 2. 스터디장 지원 (aistudy-leader.gpters.org)

### 기본 정보
- **URL**: https://aistudy-leader.gpters.org/
- **역할**: AI 스터디 리더(스터디장) 지원 페이지
- **플랫폼**: 정적 페이지 + JavaScript 렌더링 (별도 호스팅)
- **채팅**: ChannelIO 플러그인 활성 (키: `9ea17641-9c4d-4634-ae70-daa39c0b9550`)

### 페이지 구조
- 대부분의 콘텐츠가 JavaScript로 동적 렌더링
- 정적 HTML에서 추출 가능한 정보 극히 제한적
- ChannelIO 채팅 위젯으로 실시간 문의 가능

### 추정 콘텐츠 (런타임 확인 필요)
- 스터디장 역할 소개
- 지원 자격/조건
- 혜택 (수익 분배, 커뮤니티 지원 등)
- 지원 폼 또는 외부 신청 링크

## 3. B2B 기업교육 (b2b.gpters.org)

### 기본 정보
- **URL**: https://b2b.gpters.org/
- **역할**: 기업 대상 AI 교육 프로그램 랜딩 페이지
- **플랫폼**: Framer (노코드 웹 빌더)

### 디자인
- **테마**: 다크 (#171717 배경)
- **강조색**: 오렌지 (#ff4d00) - 메인 사이트(#eb5a10)와 유사하나 다름
- **폰트**: Inter (메인 사이트와 다른 폰트 체계)
- **반응형**: 3단계 브레이크포인트 (데스크톱 1200px+, 태블릿 810-1199px, 모바일 810px-)

### 페이지 구조 (추정)
1. 헤더 & 로고
2. 히어로 섹션 - 서비스 소개
3. 프로그램/교육과정 목록
4. 기업 고객 로고/사례
5. 후기/성과
6. 문의 폼 (CTA)

### 한계
- Framer 기반 동적 렌더링으로 구체적 콘텐츠(프로그램명, 가격, 후기 등) 정적 크롤링 불가

## 4. 생태계 도메인 구조

```
gpters.org (메인 - Bettermode)
├── www.gpters.org/home          ── 커뮤니티 홈
├── www.gpters.org/ai-study-list ── AI 스터디 목록
├── www.gpters.org/events        ── 이벤트
├── www.gpters.org/{spaceSlug}   ── 각 게시판
│
├── aistudy-leader.gpters.org    ── 스터디장 지원 (별도 호스팅 + ChannelIO)
├── b2b.gpters.org               ── 기업교육 (Framer)
│
└── 외부 서비스
    ├── Notion                   ── 이용약관
    ├── days.ai                  ── 개인정보처리방침
    ├── Stibee                   ── 뉴스레터
    └── Streamlit                ── AI토크 신청
```

## 핵심 발견

### 1. 플랫폼 파편화
- 메인 사이트(Bettermode), 기업교육(Framer), 스터디장 지원(별도), 약관(Notion), 개인정보(days.ai)
- 최소 5개 플랫폼에 걸쳐 서비스가 분산
- 사용자 입장에서 일관성 없는 경험

### 2. 디자인 불일치
- 메인: #eb5a10 오렌지 + 시스템 폰트 + 라이트 테마
- B2B: #ff4d00 오렌지 + Inter 폰트 + 다크 테마
- 스터디장 지원: 별도 디자인
- 브랜드 일관성 부재

### 3. 수익 전환 경로 분석

```
방문자 → [메인 홈]
  ├── 상단 배너 → AI 스터디 목록 → 스터디 신청 (내부)
  ├── GNB "AI스터디" → AI 스터디 목록 → 스터디 신청 (내부)
  ├── GNB "기업교육" → b2b.gpters.org (외부, 새 탭)
  ├── GNB "스터디장 지원" → aistudy-leader.gpters.org (외부, 새 탭)
  └── 사이드바 배너 → CTO 워크샵 (링크 미확인)
                    → B2B 기업교육 (외부)
```

- 핵심 전환(스터디 신청)은 내부에서 처리
- B2B/스터디장 지원은 외부로 이탈 → 전환 추적 어려움

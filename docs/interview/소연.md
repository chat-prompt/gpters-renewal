# 소연 인터뷰

> 역할: 마케터 (콘텐츠 마케팅)
> 인터뷰 일시: 2026-02-25

---

## 역할별 질문

### 1. 하루 루틴
- 로그인 → 사례 게시글 작성 → 업로드
- 로그인 → 공간 설정 → 띠배너 변경
- 주로 "포스트 - 최신 AI 소식" 또는 "사례 게시글 - 바이브 코딩" 등 맞는 게시판에 글을 올림

### 2. 콘텐츠 제작/관리
- 스터디 판매 기간이 되면 소구 포인트에 맞게 공간을 리뉴얼하는 작업을 진행
- 콘텐츠 마케터로서는 게시글을 작성하고 SEO에 잘 걸리게 하는 것이 주요 업무
- 글을 올릴 때 어떤 스페이스에 올려야 할지 고민될 때가 있음

### 3. 유입과 전환
- 직접 유입 또는 구글 검색이 가장 많음
- 유입 경로: AI 관련 검색어에서 지피터스 게시글 클릭 → 글 읽기 → 회원가입 → 뉴스레터나 홍보 메시지 받아보기 → 스터디 구매
- **이탈 포인트 1**: 게시글 확인 후 회원가입으로 이어지는 것이 더딤
- **이탈 포인트 2**: 상품 상세 페이지까지는 가지만 전환(구매)이 잘 안 됨

### 4. 뉴스레터/캠페인
- 보통 지피터스 게시글로 랜딩을 시킴
- 게시글 가독성이 좋지 않다고 느낌
- 다른 사이트들에 비해 가독성이 좋은 편이 아니고, 모바일도 친화적이지 않음

### 5. 내부 운영 도구
- GA, 스티비, 믹스패널, 구글 시트, 슬랙 등 다양한 외부 도구를 사용
- (상세 용도는 생략)

### 6. 성과 측정과 데이터
- 믹스패널 데이터를 기반으로 콘텐츠/캠페인 성과를 확인
- 추가로 필요한 데이터는 특별히 없음

### 7. 브랜드와 외부 인식
- 브랜딩이 더 필요하다고 느낌
- 디자인 일관성이 부족하게 느껴짐

## 공통 질문

### 카테고리/메뉴 구조
- 분류가 너무 많거나 직관성이 떨어진다고 생각함

### 잘 안 쓰는 기능/페이지
- 광고 배너 같은 게 더 잘 보였으면 좋겠음 (현재 광고 지면 활용이 부족)

### 모바일
- 모바일로 사이트에 접속하지 않음 (PC만 사용)

### 가장 불편한 점 3가지
1. **게시글의 가독성**
2. **배너 같은 광고 지면의 활용**
3. **게시판 분류 체계**

## 핵심 인사이트 요약

1. **SEO 콘텐츠가 핵심 유입 채널**: 구글 검색을 통한 게시글 유입이 가장 큰 비중을 차지하며, SEO 최적화가 마케터의 주요 업무 중 하나
2. **게시글 → 회원가입 전환 병목**: 게시글을 읽고 나서 회원가입으로 이어지는 과정이 더딤 - 게시글 내 회원가입 유도 장치가 부족할 가능성
3. **상품 상세 → 구매 전환 병목**: 상품 상세 페이지까지 도달하지만 실제 구매로 이어지지 않음 - 상품 페이지의 설득력 또는 구매 경험에 문제가 있을 수 있음
4. **가독성이 콘텐츠 마케팅의 발목을 잡고 있음**: 뉴스레터 등에서 게시글로 랜딩시키는데, 게시글 가독성이 타 사이트 대비 부족하여 콘텐츠의 효과를 떨어뜨리고 있음
5. **스페이스 분류 체계의 혼란**: 게시글 작성 시 어떤 스페이스에 올려야 할지 고민되는 상황이 발생 - 분류가 너무 많고 직관적이지 않음
6. **광고 지면 활용 부족**: 배너 등 광고 지면이 잘 보이지 않아 마케팅 활용도가 낮음
7. **디자인 일관성 부족으로 브랜딩 약화**: 사이트 전반의 디자인 일관성이 부족하여 외부에서 브랜드 인식이 약할 수 있음

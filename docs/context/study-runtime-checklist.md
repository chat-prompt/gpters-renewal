## 모집기간 (수강신청 오픈 후) — 스터디장 & 임직원 안내

> **트리거**: 수강신청 오픈일 (`모집시작일`)
> **담당**: 뽀짝이 + 연권님

### 수강신청 오픈 당일

#### 스터디장 안내 (카톡방: "지피터스 {N}기 스터디장")
- [ ] **지인초대 할인쿠폰 안내** 발송
  - 스터디장 1인당 2명 초대 가능
  - (슈퍼)얼리버드 + 추가 10만원 할인
  - AI스터디 처음 참여하는 지인만 가능
  - 초대코드 확인: https://portal.gpters.org/me/invites
  - 📝 문구 템플릿: `context/templates/leader-coupon-notice.md`
- [ ] **버디 등록 안내** 발송
  - 등록 기한 명시 (캘린더에서 확인)
  - 먼저 결제 → 수강신청 → 버디 폼 제출
  - 사례글 2회 작성 시 100% 환급(100원 제외)
  - 30명 정원 선착순
  - 폼: https://ai.gpters.org/register_buddy
  - 📝 문구 템플릿: `context/templates/leader-buddy-notice.md`

#### 임직원 안내 (Slack: #01-공지-풀타임)
- [ ] **임직원 무료등록 + 지인쿠폰 안내** Slack 공지 발송
  - 뽀짝이 멘션으로 무료초대 가능 (무료초대 스킬)
  - 임직원 지인초대: 인당 5명, 10만원 추가할인
  - 📝 문구 템플릿: `context/templates/employee-notice.md`

### 날짜 확인 포인트
- 슈퍼얼리버드 마감일 → 구글 캘린더 (`support@gpters.org`)
- 얼리버드 마감일 → 구글 캘린더
- 버디 등록 마감일 → 구글 캘린더
- 모집 마감일 → Airtable 기수관리 테이블

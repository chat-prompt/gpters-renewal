## D-28~24 (4주 전) — 스터디장 선발 & 사전판매

### 스터디장 선발 안내
- [ ] 선발 결과 안내 (문자/이메일) — `스터디장선발회신일`
- [ ] 스터디장 회신 확인 (참여 의사)
- [ ] 확정 스터디 리스트 테이블에 레코드 생성
- [ ] 스터디장 OT 일정 잡기

### 사전판매 시작
- [ ] 사전판매 오픈 (`사전판매시작일`)
- [ ] CMS 사전판매 배너 등록
- [ ] 홈 띠배너 설정
- [ ] 이메일 캠페인 발송 (뉴스레터 구독자, 대기자)
- [ ] 스레드/카톡 이벤트 기획

### 스터디장 온보딩
- [ ] 스터디장 카톡방 개설 (또는 기존 방에 초대)
- [ ] 상세페이지 작성 안내 (GPTs 상세페이지 제출 → 수정 → 최종제출)
- [ ] AI토크 일정 조율
- [ ] 협업 파트너 도구 무료 플랜 안내

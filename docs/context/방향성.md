# 이슈

- 사용자로써 지피터스 방문 시의 문제
    - 게시글이 관심이 있어 들어왔는데, 내 수준에 맞는, 내가 지금 관심있는 다른 콘텐츠, 행사, 이벤트를 발견하기 어렵다.
- 스터디 참여자로써 지피터스 사용의 문제
    - 내가 관심 있는 글을 발견하기 어렵다
    - 지피터스에서의 활동 (게시물 작성, 스터디 참여, 발표 등)을 내 포폴이나 경력으로 보여지기 어렵다
- 운영에 베터모드 사용 시의 이슈 - 사용 중인 팀원들의 이야기 들어봐야 함

# 해결책 - 사이트 리뉴얼

## 결과물

- 위 이슈를 해결하는 지피터스 사이트의 유저가 대면하는 프론트엔드의 기획
    - 기획안 = (1) 기능 및 유저 시나리오가 정리된 문서 → (2) 스크린 와이어프레임
    - 방문객이 관심이 있어하는 것을 바로 발견하고, 콘텐츠 소비하고, 커뮤니티 (채팅방) 참여 혹은 뉴스레터 구독할 수 있는 곳
    - 지피터스 스터디에 참여했던 이력이 세부적으로 잘 드러나는 나의 AI 이력서 페이지
    - 사람들이 작성했던 AI 게시물이 검색 엔진 및 LLM 대화에 잘 노출되게 최적화된 페이지
    - 게시판 및 스터디 및 커뮤니티 운영에 필요한 기능이 포함된 사이트

## 기획 방법

1. 현재 사이트의 구성 분석 및 사용자 시나리오 작성
2. 다혜/연권/재호/소연님이 현재 사이트를 활용하는 시나리오 수집
3. 수집된 시나리오 바탕으로 리뉴얼 사이트 개선안 만들기
4. 만들어진 사이트 개선안에서 새로운 유저 (운영진) 사용 시나리오 작성 및 기획안 검증

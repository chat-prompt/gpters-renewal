## 탐색 (`/explore/feed`)

- [ ] 게시글 앞 숫자+화살표(투표 UI)가 직관적이지 않음 — 중
- [ ] 게시글이 "글"처럼 느껴지지 않음. 본문 미리보기(프리뷰 텍스트)가 노출되어야 함 — 상
- [ ] 블록형 카드 레이아웃이 답답함. 요즘IT/Medium/Branch처럼 리스트형으로 가독성 개선 필요 — 상
- [ ] 정렬 기준(인기/최신/추천)이 탭으로 되어 있는데, 우측 드롭다운이 더 깔끔할 듯 — 중
- [ ] 태그 필터가 껍데기만 있음. 선택된 태그만 표시하는 UI인데 태그를 선택하는 방법(드롭다운/검색)이 없어서 작동 불가 — 상
- [ ] 카테고리 필터 선택해도 게시글이 필터링되지 않음. 프로토타입이라도 mock 필터링 로직 필요 — 상
- [ ] 정렬 옵션 "인기"와 "추천"의 차이가 불명확. 인기/최신 2개로 충분 — 중
- [ ] 카테고리에 디자인(Midjourney, DALL-E, Figma AI 등) 영역이 없음. AI 활용법에 묻힘 — 상
- [ ] 카테고리에 미디어/크리에이티브(영상, 이미지, 음악 — Runway, Suno, Pika 등) 영역이 없음 — 상
- [ ] "AI 활용법" 카테고리명이 너무 포괄적/치우침. 다른 카테고리도 결국 AI 활용법인데 구분이 모호 — 중
- [ ] "AI 뉴스"보다 "트렌드"가 더 적합할 수 있음. 뉴스는 단순 전달, 트렌드는 흐름+인사이트 포함 — 하

# ai-study.gpters.org 페이지 스크린샷 설명

## 파일 목록

### 01_home.png
- **URL**: https://ai-study.gpters.org/
- **페이지명**: 홈 (대시보드)
- **주요 기능**:
  - 오늘 진행되는 스터디 위젯 (스터디 시작 일정 표시)
  - 내 스터디 바로가기 (과제 진행률, 북마크 수, 과제/다시보기/게시글 빠른 이동)
  - AI랑 찐친되기 챌린지 (기간, 전체 인증 수, D-day 표시)
  - 활동왕 TOP 10 (게시글 많이 쓴 유저 랭킹)

### 02_studies_top.png
- **URL**: https://ai-study.gpters.org/studies
- **페이지명**: 전체 스터디 (상단)
- **주요 기능**:
  - 요일 필터 버튼 (전체/화/수/목)
  - 카테고리 드롭다운 필터 (전체/개발&자동화/업무&비즈니스/콘텐츠&지식)
  - 내 스터디 필터 버튼
  - 스터디 카드 그리드 (썸네일, 이름, 요일, 카테고리, 스터디장, 단톡방 버튼, 캘린더 버튼)
  - 총 18개 스터디

### 03_studies_bottom.png
- **URL**: https://ai-study.gpters.org/studies (스크롤 하단)
- **페이지명**: 전체 스터디 (하단)
- **주요 기능**:
  - 스터디 목록 하단부 (Manus스킬, 소셜임팩트, 교육앱만들기, ChatAPT, 내살자동화, 전자책출판)

### 04_notice.png
- **URL**: https://ai-study.gpters.org/notice
- **페이지명**: 공지사항
- **주요 기능**:
  - 공지사항 목록 페이지 (현재 등록된 공지 없음 - 빈 상태)
  - 홈으로 돌아가기 버튼

### 05_replay.png
- **URL**: https://ai-study.gpters.org/replay
- **페이지명**: 다시보기
- **주요 기능**:
  - 링크 다운로드 버튼 (다시보기 권한 있는 주차만 다운로드 가능)
  - 스터디 선택 드롭다운 (전체 스터디)
  - 내 스터디만 / 시청 가능한 영상만 토글 필터
  - 강의 목록 (카테고리별 그룹핑, 영상 수 표시)
  - 우측 유튜브 영상 플레이어 내장
  - AI토크기간/공통일정 분류 태그, 시청 가능 여부, 줌채팅 다운로드 버튼

### 06_posts_list.png
- **URL**: https://ai-study.gpters.org/posts
- **페이지명**: 게시글 모아보기 (목록)
- **주요 기능**:
  - 스터디 선택 드롭다운, 주차 선택 드롭다운
  - 내 스터디만 / 내 글만 토글 필터
  - 좌측: 게시글 목록 (주차 뱃지, 스터디명, 날짜, 제목, 작성자, 북마크 아이콘)
  - 우측: 게시글 선택 전 안내 문구 표시

### 07_posts_detail.png
- **URL**: https://ai-study.gpters.org/posts (게시글 선택 후)
- **페이지명**: 게시글 모아보기 (상세)
- **주요 기능**:
  - 좌측 목록에서 게시글 클릭 시 우측 패널에 내용 표시 (2-panel 레이아웃)
  - 북마크 버튼, 원본 보기 버튼
  - 게시글 내용 (제목, 작성자, 날짜, 본문)

### 08_assignment_info.png
- **URL**: https://ai-study.gpters.org/assignment
- **페이지명**: 나의 학습 > 스터디 정보 탭
- **주요 기능**:
  - 내 스터디 썸네일, 기수/카테고리, 스터디명, 스터디장, 요일/시간 정보
  - 단톡방 버튼, 캘린더 버튼
  - 스터디 공지사항 섹션
  - 우측 캘린더 (스터디 일정 표시, 내 스터디/공통세션 색상 구분)
  - 학습 가이드 (다시보기 안내, 과제 제출 마감 및 발표 규칙, 수료 기준)

### 09_assignment_status.png
- **URL**: https://ai-study.gpters.org/assignment?tab=assignment
- **페이지명**: 나의 학습 > 과제 현황 탭
- **주요 기능**:
  - 수료 현황 카드 (D-day, 제출 횟수, 진행중 상태, 마감 안내)
  - 주차별 과제 목록 (2주차/3주차/4주차, 제출 내역, 곧 열림 상태)
  - 우측: 선택한 주차 과제 상세 (마감일, 과제 안내, 제출 내역)
  - 과제 제출하러 가기 버튼

### 10_assignment_bookmark.png
- **URL**: https://ai-study.gpters.org/assignment?tab=bookmark
- **페이지명**: 나의 학습 > 내 북마크 탭
- **주요 기능**:
  - 북마크한 게시글 목록 (현재 비어있음)
  - 게시글 보러가기 버튼

### 11_404.png
- **URL**: https://ai-study.gpters.org/[존재하지 않는 경로]
- **페이지명**: 404 에러 페이지
- **주요 기능**:
  - 잘못된 경로 접근 시 표시
  - 좌측 사이드바는 유지됨 (로그인 상태 유지)

## 사이트 전체 구조 요약

### 사이드바 메뉴
- 홈 (/)
- 전체 스터디 (/studies)
- 공지사항 (/notice)
- 다시보기 (/replay)
- 게시글 (/posts)
- 나의학습 (/assignment) - 스터디정보/과제현황/내북마크 탭

### 공통 UI 요소
- 좌측 고정 사이드바 (사이드바 접기 버튼 있음)
- 하단 유저 정보 (이름, 기수/직함)
- 로그아웃 버튼

### 외부 연결 (ai-study 내부가 아님)
- 스터디 상세보기 → https://www.gpters.org/study/...
- 과제 제출 → https://www.gpters.org/
- 게시글 원본 보기 → https://www.gpters.org/

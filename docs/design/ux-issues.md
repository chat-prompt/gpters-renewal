## GNB (네비게이션)

- [x] 검색이 아이콘만 있어 검색바로 인식되지 않음 — 상 → GNB에 Medium 스타일 검색바 (rounded-full, placeholder 텍스트) 추가 (2026-03-05)
- [x] 글쓰기 버튼이 페이지별로 분산되어 있거나 없음 — 상 → GNB 우측에 글쓰기 버튼 상시 노출 (PenSquare 아이콘) (2026-03-05)
- [x] GNB가 max-width로 제한되어 좌우 여백이 낭비됨 — 중 → 전체 너비 3-column 레이아웃으로 변경 (2026-03-05)

## 탐색 (`/explore/feed`)

- [x] 게시글 앞 숫자+화살표(투표 UI)가 직관적이지 않음 — 중 → PostCard에서 투표 UI 제거, 하트 아이콘(좋아요)으로 교체 (2026-03-04)
- [x] 게시글이 "글"처럼 느껴지지 않음. 본문 미리보기(프리뷰 텍스트)가 노출되어야 함 — 상 → excerpt line-clamp-3 노출, Medium 스타일 리스트형 (2026-03-05)
- [x] 블록형 카드 레이아웃이 답답함. 요즘IT/Medium/Branch처럼 리스트형으로 가독성 개선 필요 — 상 → Medium 스타일 리스트형, 보더 제거 → divide-y 디바이더만 사용 (2026-03-05)
- [x] 정렬 기준(인기/최신/추천)이 탭으로 되어 있는데, 우측 드롭다운이 더 깔끔할 듯 — 중 → ghost 스타일 드롭다운, 게시글 목록 위 좌측 정렬 (2026-03-05)
- [x] 태그 필터가 껍데기만 있음. 선택된 태그만 표시하는 UI인데 태그를 선택하는 방법(드롭다운/검색)이 없어서 작동 불가 — 상 → 전체 태그 토글 pill UI로 교체 (2026-03-04)
- [x] 카테고리 필터 선택해도 게시글이 필터링되지 않음. 프로토타입이라도 mock 필터링 로직 필요 — 상 → mock 필터링 로직 구현 (카테고리/태그/정렬) (2026-03-04)
- [x] 정렬 옵션 "인기"와 "추천"의 차이가 불명확. 인기/최신 2개로 충분 — 중 → 인기/최신 2개로 축소 (2026-03-04)
- [x] 카테고리에 디자인(Midjourney, DALL-E, Figma AI 등) 영역이 없음. AI 활용법에 묻힘 — 상 → 디자인 카테고리 추가 (2026-03-04)
- [x] 카테고리에 미디어/크리에이티브(영상, 이미지, 음악 — Runway, Suno, Pika 등) 영역이 없음 — 상 → 미디어 카테고리 추가 (2026-03-04)
- [ ] "AI 활용법" 카테고리명이 너무 포괄적/치우침. 다른 카테고리도 결국 AI 활용법인데 구분이 모호 — 중
- [x] "AI 뉴스"보다 "트렌드"가 더 적합할 수 있음. 뉴스는 단순 전달, 트렌드는 흐름+인사이트 포함 — 하 → 뉴스→트렌드 변경 완료 (2026-03-04)
- [x] 메인 영역과 사이드바의 태그 필터가 중복됨 (TagFilter + 추천 토픽) — 중 → 메인 TagFilter 제거, 사이드바 추천 토픽만 유지 (2026-03-05)
- [x] 에디터 픽이 제목만 있어 너무 밋밋함 — 중 → 아바타+작성자, 제목(bold), 날짜 구조로 확장 (2026-03-05)
- [x] 추천 작성자(Who to Follow)가 단순 텍스트 목록 — 중 → X/Twitter 스타일 프로필 카드 (40px 아바타, @username, bio, Follow 버튼) (2026-03-05)
- [x] 사이드바 섹션 간 border-b 구분선이 Medium과 다름 — 하 → 구분선 제거, pb-8 간격만으로 시각 분리 (2026-03-05)
- [x] PostCard 텍스트 위계가 약함 (제목과 본문 크기 차이 부족) — 상 → 제목 text-xl font-extrabold, 본문 text-base, 메타 text-xs (2026-03-05)
- [x] PostCard 썸네일이 정사각형(w-28 h-28)이라 Medium과 다름 — 하 → 직사각형 w-28 h-20으로 변경 (2026-03-05)

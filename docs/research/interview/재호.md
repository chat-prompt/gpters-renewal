> 인터뷰 일시: 2026-02-25

## 역할별 질문

### 1. 하루 루틴

- AI 설명회가 예정되어 있었고, 이를 이벤트 상단에 잘 보여지게 교육 담당자 연권님과 함께 조정하였음
- 어드민 페이지에 가서 메뉴 설정을 수정함
- 메타 광고 소재 제작을 위해 각 이벤트 페이지를 접속하여 세션의 내용을 파악하고 URL을 확인하여 UTM을 붙이는 작업을 수행함
- AI 토크 상세페이지를 확인하여 진행되는 내용을 열람함

### 2. 콘텐츠 제작/관리

- 어떤 스페이스를 들어가서 수정해야 원하는 UI가 변경되는지 찾기가 어려움
- 각 섹션에서 타이틀이나 문구 등을 자유롭게 수정하는 데 제약사항이 많다고 느낌

### 3. 유입과 전환

- 검색 유입이 가장 많은 편
- 뉴스레터나 광고 등은 부수적인 유입 경로
- 정확한 데이터는 믹스패널을 참고하는 것이 좋을 것
- 유입 후 이탈이 많은 단계에 대해서는 명확히 파악하지 못하고 있음

### 4. 뉴스레터/캠페인

- 뉴스레터나 마케팅 캠페인 진행 시 사이트를 활용하는 경우는 거의 없음

### 5. 내부 운영 도구

- **스티비**: 뉴스레터 발송
- **구글시트**: UTM 생성
- **메타 광고 관리자**: 메타 광고 운영
- **카카오모먼트**: 카카오 광고 운영
- 사이트와 외부 도구의 연계에 대한 불편함은 특별히 없음

### 6. 성과 측정과 데이터

- 믹스패널 수치를 참고하거나 오가닉 수치를 확인함
- 현재 볼 수 있는 데이터로 충분하다고 느낌

### 7. 브랜드와 외부 인식

- 외부에서 지피터스를 어떻게 인식하는지에 대해서는 잘 모르겠다고 함

## 공통 질문

### 카테고리/메뉴 구조

- 카테고리가 너무 많다고 생각함
- 모든 메뉴 앞에 붙어 있는 이모지가 오히려 혼란스럽게 만든다고 느낌

### 잘 안 쓰는 기능/페이지

- 가입인사 게시판이 의미 있는지 의문

### 모바일

- 모바일로 사이트를 거의 사용하지 않음 (PC 위주)

### 가장 불편한 점 (2가지)

1. **게시판 수정 방식이 불편함** - 어떤 스페이스를 수정해야 원하는 UI가 바뀌는지 찾기 어렵고, 수정 자체에도 제약이 많음
2. **상품/쿠폰 관리가 외부 도구 의존** - 상품을 생성하고 쿠폰을 연결하는 과정이 리툴(Retool)이라는 별도 도구를 사용해야 해서 불편함

## 핵심 인사이트 요약

1. **어드민 UX의 직관성 부족**: 원하는 UI를 변경하기 위해 어떤 스페이스를 수정해야 하는지 찾기 어려움. 스페이스와 실제 사이트 UI 간의 매핑이 불명확함
2. **콘텐츠 수정 제약**: 각 섹션의 타이틀이나 문구를 자유롭게 수정하는 데 제약이 많아, 마케터가 빠르게 콘텐츠를 업데이트하기 어려움
3. **상품/쿠폰 관리의 외부 도구 의존**: 상품 생성-쿠폰 연결 과정이 리툴(Retool)에 의존하고 있어 워크플로우가 분산됨
4. **과도한 카테고리와 이모지 혼란**: 17개 이상의 스페이스와 모든 메뉴에 붙은 이모지가 정보 탐색을 방해하고 혼란을 줌
5. **UTM 관리의 외부 의존**: UTM 생성을 구글시트로 별도 관리하고 있으며, 광고 소재 제작 시 사이트와 외부 도구를 오가며 작업해야 함
6. **가입인사 게시판의 존재 의문**: 마케터 관점에서 가입인사 게시판의 실질적인 가치에 의문을 가지고 있음

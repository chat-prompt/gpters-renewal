## 1. 홈 (/) - 콘텐츠 디스커버리 중심

### 설계 의도
- 방문자가 즉시 "여기에 내가 찾는 것이 있다"고 느끼게 함
- 인터뷰: 신규 회원 "뭐가 이렇게 많지?" 반응 → 정보량 제한, 우선순위 명확화

## 4. 스터디 상세 (/study/{slug}) - 전환 최적화된 스터디 소개

### 설계 의도
- 마케터 인터뷰: "상품 상세 페이지까지는 가지만 전환(구매)이 잘 안 됨"
- 패스트캠퍼스 참고: 히어로 → 소구 포인트 → 커리큘럼 → 사회적 증거 → CTA 구조
- Sticky CTA 버튼으로 어느 위치에서든 바로 신청 가능
- 스터디장 프로필 연결로 신뢰도 확보

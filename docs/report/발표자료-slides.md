# 네비게이션 구조

<br>

### GNB (Global Navigation Bar)

| 메뉴 | 역할 | 해결하는 문제 |
|------|------|-------------|
| **탐색** | 콘텐츠 피드 + 필터 | "콘텐츠 발견이 어렵다" |
| **스터디** | 모집/진행 중 목록 | "모집이 진행 중인지 안 보인다" |
| **커뮤니티** | 자유 게시판, Q&A, 이벤트 | 커뮤니티 성격 강화 |
| **AI 이력서** | 프로필 + 이력 + 포트폴리오 | "활동을 경력으로 보여주기 어렵다" |

<br>

### 핵심 원칙
- 모든 콘텐츠 **3클릭 이내** 도달
- 게시글 URL `/posts/{slug}` (카테고리 변경해도 URL 불변 → SEO 안정)

---

# 전체 사이트맵

<br>

```
홈 (/)
├── 탐색        → 6개 카테고리 탭 + 태그 필터 + 투표 정렬
│   └── 게시글 상세   → SEO 랜딩, 전환 CTA 포함
│
├── 스터디      → 모집 중 우선 노출
│   ├── 상세        → 전환 최적화 (패스트캠퍼스 참고)
│   └── 내 스터디   → LMS 바로가기 (1클릭)
│
├── 커뮤니티    → 자유 게시판 / Q&A / 이벤트
│
├── AI 이력서   → 스킬맵 + 스터디 이력 + 포트폴리오
│
├── 글쓰기      → 임시저장 지원
│
└── 관리자      → 최대 2단계 (대시보드 / 게시글 / 스터디 / 배너)
```

---

# Flow 1: 신규 방문자 → 회원 전환

<br>

```
구글에서 "Claude 마케팅 자동화" 검색
              ↓
      게시글 페이지 유입 (SEO)
              ↓
      가독성 높은 본문 읽기 (720px, 행간 1.8)
              ↓
     ┌────────┴────────┐
     ↓                 ↓
  인라인 CTA         관련 스터디 배너
  "도움되었나요?      "이 주제를 깊이
   가입하세요"         배우고 싶다면?"
     ↓                 ↓
  회원가입            스터디 상세
```

**개선**: 게시글 내 전환 장치 배치 (기존에는 없었음)

---

# Flow 2: 회원 → 콘텐츠 탐색

<br>

```
홈 방문
  ├── 히어로 배너: 모집 중인 스터디 확인
  ├── 인기 콘텐츠: 투표수 기반 큐레이션 (Reddit식)
  └── 카테고리 탐색 클릭
              ↓
      탐색 페이지 (/explore)
              ↓
     ┌────────┴────────┐
     ↓                 ↓
  카테고리 탭 선택   태그 필터 조합
  (6개 중 택 1)     (#Claude #입문)
     └────────┬────────┘
              ↓
      인기순/최신순 정렬 → 게시글 목록 → 투표/북마크/읽기
```

**개선**: 30개 카테고리 → 6개 탭 + 태그로 단순화

---

# Flow 3: 회원 → 스터디 구매

<br>

```
GNB "스터디" 클릭 (1클릭)
          ↓
    스터디 목록 (모집 중 우선)
          ↓
    스터디 상세 페이지
          ↓
  ┌───────┼───────┬───────┐
  ↓       ↓       ↓       ↓
히어로  커리큘럼  스터디장  수료생 후기
(모집률) (아코디언) (프로필)  (사회적 증거)
  └───────┼───────┴───────┘
          ↓
  Sticky CTA: "잔여 8석 | 스터디 신청하기"
          ↓
    결제 → 내 스터디 → LMS
```

**개선**: 모집 진행률로 긴급성 유발 + 후기로 사회적 증거

# Flow 4: 스터디 참여자 → 학습

<br>

<div class="columns">
<div class="col">

### Before (4단계)

2. 내 AI스터디 클릭
3. 로그인
4. LMS 페이지 이동

### After (1~2단계)
- **방법 1**: 프로필 드롭다운 → "내 스터디" (최상단)
- **방법 2**: GNB "스터디" → 내 스터디

활동이 **AI 이력서에 자동 반영**
- 스터디 수료 이력
- 작성한 사례글
- 스킬맵 업데이트

</div>
</div>

# Flow 5: 운영자 → 관리 작업

<br>

<div class="columns">

### After

```

      ↓
  오늘 요약 + 빠른 작업
      ↓
 ┌────┼────┬────┐
 ↓    ↓    ↓    ↓
게시글 스터디 배너 회원
관리   관리  관리 관리
```

- 배너: 드래그앤드롭 순서 변경
- 스터디: 목록에서 상태/토글 원클릭

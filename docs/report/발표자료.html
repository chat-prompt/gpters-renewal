User Action Flow

핵심 유저 시나리오

신규 방문자: 바이브 코딩 → 가입
Google 검색 → 게시글 열람 → 관련 콘텐츠 추천 → 회원가입 유도 CTA → 관심사 온보딩 → 뉴스레터 구독 → 스터디 탐색
스터디 참여자: 학습 → 포트폴리오
로그인 → 내 대시보드 → 1클릭 스터디 접근 → 학습 후 글 작성 → 자동 임시저장 → 게시 → AI 이력서 반영
운영자: 스터디 관리 + 홈 수정
로그인 → 관리자 대시보드 → 오늘 할 일 확인 → 스터디 상태 변경 → CMS 페이지 편집 → 배너/상품 관리
마케터: 콘텐츠 → 전환 추적
SEO 콘텐츠 작성 → 카테고리 선택 (6개) → CTA 배치 → 게시 → 성과 분석 → 광고 지면 관리

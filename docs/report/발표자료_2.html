문제점 1/2

현재 사이트의 핵심 문제 (1~7)

1
DM 시스템 완전 미작동
자동 DM과 실제 DM 혼재 → 중요 메시지 누락. 이메일 알림 클릭해도 리디렉션 안 됨.
"누군가 DM을 보내면 완전 누락하는 경우가 많음" — 다혜
2
게시글 공간이동 불가 → 웹훅 중복 연쇄
잘못된 게시판에 올리면 삭제→재게시 필수 → 웹훅 2회 발동 → n8n 자동요약 2번 실행.
"자주 발생하는 문제" — 다혜
3
카테고리 ~30개, 분류 기준 혼재
도구별/분야별/시간별 기준 섞임. 자주 쓰는 건 3~4개뿐.
"이모지가 오히려 혼란스럽다" — 재호 · "통합 필요" — 다혜
4
게시판이 "과제 제출용"으로만 작동
가벼운 피드형 포스트 없음. 사례게시글 포맷이 무거워 진입장벽 높음.
"가벼운 포스트 형식이 필요하다" — 다혜
5
Admin 스페이스-UI 매핑 불명확
어떤 스페이스를 수정해야 원하는 UI가 바뀌는지 모름. 홈 수정에 7~8단계 필요.
"찾기가 어려움" — 재호 · "여러 번 헤맴" — 신연권
6
검색/정렬/북마크 기능 부족
원하는 글 찾기 어려움, 정렬 기준 불명확, 북마크 기능 사라짐, 영상 색인 없음.
유저 불만 24건 수집 — 다혜
7
게시글 가독성 부족 + 전환 유도 부재
타이포그래피/여백 부족. 게시글 → 회원가입 전환 장치 없음.
"가독성이 좋지 않다", "전환이 더딤" — 소연
문제점 2/2

현재 사이트의 핵심 문제 (8~14)

8
임시저장 기능 부재
작성 중 페이지 이탈 시 내용 전부 소실. 외부 에디터에서 작성 후 복붙하는 우회 방식.
"외부에서 작성 완료 후 복사 붙여넣기" — 신연권
9
스터디 접근 경로 4단계
프로필 → 내 AI스터디 → 로그인 → LMS 이동. 신규 참여자 "스터디 어디서 듣나요?" 빈번.
"어디로 들어가야 하나요?" 반복 질문 — 신연권
10
상품/쿠폰 관리 Retool 의존
상품 생성·쿠폰 연결을 별도 외부 도구(Retool)로 처리해야 함.
"불편함" — 재호
11
콘텐츠 수정 자유도 부족
각 섹션 타이틀/문구를 자유롭게 수정하기 어려움.
"제약사항이 많다" — 재호
12
프로필이 포트폴리오로 기능하지 못함
스터디 참여 이력, 사례글이 한눈에 안 보임. AI 활용 역량을 외부에 보여줄 방법 없음.
"AI계의 LinkedIn이 되어야 한다" — 다혜
13
게시/수정 버튼 위치 불일치
게시 버튼 최하단, 수정 버튼 최상단. 긴 글에서 스크롤 압박. 스터디 종류 오기입 발생.
유저 불만 수집 — 다혜
14
댓글 UX 불편
"1개 더 많은 답글 보기" 일일이 클릭해야 함. 전체 댓글 한 번에 보기 불가.
유저 불만 수집 — 다혜
Solution Direction

개선 방향

14개 문제에 대한 12가지 개선 방향
개선 방향 1/2

핵심 개선 방향 (1~6)

1
콘텐츠 참여 2트랙
피드 포스트 (짧은 공유, 자랑글) + 사례 게시글 (상세 활용기). 누구나 가볍게 참여 가능.
해결: #4 게시판 "과제 제출용" 문제
2
정보구조 단순화
30개 카테고리 → 6개 대분류 + 태그 시스템. 이모지 제거, 일관된 분류 기준.
해결: #3 카테고리 혼재
3
"AI계의 LinkedIn" 프로필
스터디 수료, 사례글, 발표 이력 자동 누적. AI 활용 패턴 자동 요약 + 뱃지/포인트 + 외부 공유 링크.
해결: #12 프로필 포트폴리오 부재
4
메시징 시스템 재설계
자동 알림 / 실제 DM 채널 분리. 알림 클릭 시 해당 메시지로 정확히 이동.
해결: #1 DM 시스템 미작동
5
에디터 + 임시저장
자동 임시저장(30초) + 수동 저장 + 드래프트 목록. 이탈 시 확인 다이얼로그.
해결: #8 임시저장 부재
6
스터디 접근 1클릭
로그인 후 대시보드에 "내 스터디" 즉시 노출. 4단계 → 1클릭.
해결: #9 스터디 접근 4단계
개선 방향 2/2

핵심 개선 방향 (7~12)

7
Admin UX 평탄화
7단계 → 최대 2단계 관리자 대시보드. 스페이스-UI 매핑 시각화. 상품/쿠폰 어드민 내장 (Retool 제거).
해결: #5 Admin 매핑 불명확 · #10 Retool 의존
8
게시글 가독성 + 전환 최적화
타이포그래피 시스템 (max-width 720px, line-height 1.8). 게시글 내 인라인 CTA + 관련 스터디 추천.
해결: #7 가독성 부족 · 전환 유도 부재
9
검색/탐색 강화
통합 검색 (게시글+스터디+사용자). 카테고리/태그 필터 + 다양한 정렬. 북마크 복원 + 영상 색인.
해결: #6 검색/북마크 부족
10
게이미피케이션
기여 기반 포인트 (사례글, AI토크, 모각AI 리딩). 뱃지 체계화 + 프로필 자동 반영.
동기부여 효과 큼 (다혜) → 기준 명확화 필요
11
웹훅/이벤트 정상화
멤버 셀프 게시글 이동 가능. 이벤트 중복 발동 방지 (idempotency).
해결: #2 웹훅 중복 연쇄
12
미래 확장 (로드맵)
채팅방 ↔ 게시판 양방향 (26년 2Q). 구인/구직 게시판 (안정화 이후). 모각AI 온/오프라인.
26년 Q2~ 순차 도입
